# Python And ML Tools

## Core Now

### NumPy

- Docs: https://numpy.org/doc/stable/
- Use for: Arrays, broadcasting, matrix multiplication, linear algebra, random data generation, and scratch implementations.
- First 12-week priority: `ndarray`, broadcasting, `@` matrix multiplication, `numpy.linalg`, and vectorised loss functions.

### pandas

- Docs: https://pandas.pydata.org/docs/
- Use for: Tabular data, date indexes, joins, rolling windows, shifting targets, and monthly ETF features.
- First 12-week priority: indexing, `shift`, `rolling`, `groupby`, missing data, and resampling.

### matplotlib

- Docs: https://matplotlib.org/stable/
- Use for: Residual plots, loss curves, validation-window plots, confusion matrices, and model comparison charts.
- First 12-week priority: Basic line plots, scatter plots, subplots, labels, legends, and saving figures.

### scikit-learn

- Docs: https://scikit-learn.org/stable/
- User guide: https://scikit-learn.org/stable/user_guide.html
- Linear models: https://scikit-learn.org/stable/modules/linear_model.html
- Model evaluation: https://scikit-learn.org/stable/modules/model_evaluation.html
- Use for: Reference implementations, preprocessing, metrics, model comparison, and pipelines.
- First 12-week priority: `LinearRegression`, `LogisticRegression`, `Ridge`, `Lasso`, tree models, ensembles, metrics, and validation tools.
