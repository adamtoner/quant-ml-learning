# Statistical Learning

## Core Now

### An Introduction to Statistical Learning

- Official: https://www.statlearning.com/
- Use for: Main reading-first guide to classical ML and model assessment.
- First 12-week priority: Linear regression, classification, resampling, regularisation, tree-based methods, and model assessment.
- Practical rule: Read the relevant chapter before using scikit-learn for that model.

## Later

### The Elements of Statistical Learning

- Official: https://hastie.su.domains/ElemStatLearn/
- PDF: https://hastie.su.domains/Papers/ESLII.pdf
- Use for: More mathematical treatment of statistical learning after ISL concepts are working in code.
- Priority sections: Linear methods, model assessment, regularisation, trees, boosting, and high-dimensional issues.
- Practical rule: Treat as a later-stage reference. Do not let it slow the first 12-week implementation cycle.
