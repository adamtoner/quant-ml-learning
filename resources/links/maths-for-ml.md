# Maths For ML

## Core Now

### Mathematics for Machine Learning

- Official: https://mml-book.github.io/
- PDF: https://mml-book.github.io/book/mml-book.pdf
- Use for: Linear algebra, matrix notation, projections, vector calculus, probability, and optimisation foundations.
- First 12-week priority: Matrix notation, inner products, norms, least-squares/projection material, gradients, and probability basics.
- Practical rule: Read only the sections needed for the current weekly block, then implement the related concept.

### 3Blue1Brown Essence of Linear Algebra

- Link: https://www.3blue1brown.com/topics/linear-algebra
- Use for: Visual intuition for vectors, matrices, linear transformations, eigenvectors, and projections.
- First 12-week priority: Use as a supplement when MML notation feels dry or when geometric intuition is missing.
- Practical rule: Watch selectively. Do not substitute video familiarity for derivation and implementation.

## Later

- Return to broader MML chapters after the first 12 weeks when optimisation, PCA, kernels, or deeper probability topics become necessary.
