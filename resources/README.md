# Resources

This folder is for curated learning references, not an exhaustive library. Add material only when it directly supports the current plan.

## Folder Layout

```text
resources/
├── README.md
├── downloads/          # PDFs or local files actively being used
└── links/              # Reference lists and usage guidance
```

## Rules

- Do not download files automatically.
- Only download PDFs that are actively being used in the current block.
- Keep links curated. Avoid collecting resources that are not tied to a near-term task.
- Prefer reading-first resources, then practical implementation docs.
- If a resource is not being used in the first 12 weeks, mark it as later-stage.

## First 12 Weeks

The first 12 weeks should rely mainly on:

- Mathematics for Machine Learning.
- An Introduction to Statistical Learning.
- scikit-learn documentation.
- NumPy and pandas documentation.
- Selected Dive into Deep Learning sections where they clarify optimisation or linear/logistic models.

Avoid resource hoarding. The goal is to read, implement, test, and write notes, not to build a large reference pile.
